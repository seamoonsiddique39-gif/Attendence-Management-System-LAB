Issue 1:
Title: Authentication Method Not Defined
Description: The login requirement does not specify whether users will authenticate using email, username, or student ID.
Defect Type: Ambiguity
Impact: Developers may implement inconsistent authentication methods causing confusion for users.
Suggested Fix: Clearly define the authentication method and required credentials (email, username, or student ID).
Priority: Medium
